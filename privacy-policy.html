Privacy Policy for PomodoroYT

Effective date: April 2, 2026

PomodoroYT is a Chrome extension that helps users run Pomodoro sessions on YouTube with timer controls, a floating widget, break reminders, theme settings, and session-based playback control.

1. Information PomodoroYT accesses

PomodoroYT may access limited information from the YouTube tab selected by the user, including:

  • Page URL
  • Page title
  • Basic playback state, such as whether a video is playing or paused
  • Basic page state needed to show the timer widget and keep the selected tab linked to the timer session

2. Information PomodoroYT stores

PomodoroYT stores settings locally in the user’s browser using Chrome extension storage, such as:

  • Focus, short break, and long break durations
  • Theme preference
  • Notification preference
  • Floating widget preferences
  • Current timer/session state
  • Selected locked YouTube tab information needed for the extension to function

3. How this information is used

PomodoroYT uses this information only to provide its core features, including:

  • Running the Pomodoro timer
  • Keeping one selected YouTube tab linked to the timer session
  • Displaying the floating timer widget on YouTube
  • Showing break reminders and optional notifications
  • Controlling playback behavior during focus and break sessions
  • Saving user settings between browser sessions

4. Data sharing

PomodoroYT does not sell, rent, or transfer user data to third parties.

PomodoroYT does not use user data for advertising, profiling, credit evaluation, lending, or any purpose unrelated to the extension’s single purpose.

5. Remote servers

PomodoroYT does not require users to create an account and does not send personal data to a remote server for the extension’s main functionality.

6. Notifications

If enabled by the user, PomodoroYT may show local browser notifications for timer and session changes. These notifications are generated locally by the extension.

7. Data retention

Data stored by PomodoroYT remains in the user’s browser until it is changed, cleared by the user, or removed when the extension is uninstalled.

8. Security

PomodoroYT is designed to keep its data usage limited to the minimum needed for its features. However, no software system can guarantee absolute security.

9. Children’s privacy

PomodoroYT is a general productivity extension and is not specifically directed toward children.

10. Changes to this policy

This Privacy Policy may be updated from time to time. Any changes will be posted on this page with an updated effective date.
