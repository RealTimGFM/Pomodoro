Behavior notes

What these settings affect

  • Changes are saved in localStorage, so they stay local to this browser.
  • Duration updates apply immediately to idle sessions and to future sessions in a running cycle.
  • Browser notifications stay optional and degrade gracefully if permission is denied.
  • The app uses the official SoundCloud widget in the browser instead of a custom backend media service.
  • If autoplay is blocked after a reload, the saved SoundCloud source still restores and the UI will tell you to press play once.
  • The timer still works fully even if you never load any media.
