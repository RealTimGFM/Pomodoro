Future roadmap for Pomodoro Flow

1. Provider abstraction
- Add a clean multi-provider media layer so SoundCloud, Spotify, and future providers can share one timer contract.

2. Browser extension experiments
- Explore a YouTube browser-extension version that can coordinate playback in the user's own browser context without turning the main web app back into a YouTube-first product.

3. Better playlist controls
- Add queue-aware UI around playlist position, artwork, and richer now-playing details while still keeping the timer visually dominant.

4. Shared focus rooms
- Introduce lightweight study rooms with synchronized session states and presence indicators.

5. Cloud sync
- Add optional cloud-backed persistence for tasks, settings, and session history later, without making local-first usage dependent on accounts.

6. Session analytics
- Add daily and weekly focus summaries, completion streaks, and more thoughtful progress history.

7. Mobile app
- Rebuild the experience as a dedicated mobile app once the browser-first interaction model is stable.
